### First Qu and Ans

# 1. What are some differences between interfaces and types in TypeScript?
        **Ans :** 'টাইপ' এবং 'ইন্টারফেস' উভয়ই ডেটার গঠন সংজ্ঞায়িত করতে ব্যবহত হয় ,
        *টাইপ'* : টাইপ এর ক্ষেত্রে প্রথমে টাইপ তারপর টাইপের নাম দিতে হা ,  বিভিন্ন ডাটা যেমন- ইউনিয়ন, ইন্টারসেকশন এবং সাধারণ ধরনের ডেটা (যেমনবুলিয়ান: সংখ্যা, স্ট্রিং ) সংজ্ঞায়িত করতে বেশি উপযোগী ।
        *ইন্টারফেস* :অবজেক্ট এবং ক্লাসের গঠন সংজ্ঞায়িত করা, পাবলিক API তৈরি করা। মূলত অবজেক্ট বা ক্লাসের গঠন এবং পাবলিক API সংজ্ঞায়িত করতে ব্যবহৃত হয় এবং এটি অতিরিক্ত বৈশিষ্ট্য (extended) করা যায়।

        
### Second Qu and Ans 
# 2. 

   **Ans :**  ১. ইউনিয়ন টাইপ : 
ইউনিয়ন টাইপ TypeScript-এ একটি টাইপ সিস্টেম ফিচার যা একাধিক টাইপকে একত্রিত করে। এটি বলে দেয় যে, একটি ভেরিয়েবল একাধিক টাইপের যেকোনো একটি ধারণ করতে পারে। একে | (পাইপ) চিহ্ন দিয়ে প্রকাশ করা হয়।
উদাহরন: let age: string | number;
age = 10
age = "15"

ইন্টারসেকশন টাইপ :
ইন্টারসেকশন টাইপ TypeScript-এ এমন একটি ফিচার যা দুটি বা তার বেশি টাইপকে একত্রিত করে একটি নতুন টাইপ তৈরি করে, যেখানে নতুন টাইপটি সমস্ত টাইপের সমস্ত প্রপার্টি ধারণ করবে। একে & (এম্পারস্যান্ড) চিহ্ন দিয়ে প্রকাশ করা হয়।
উদাহরন :   type User = {name:string}    
        type Product = {title: string}
        type UserProduct = User & Product
