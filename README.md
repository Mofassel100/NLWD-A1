#First Qu and Ans

#1. What are some differences between interfaces and types in TypeScript?
        Ans:'টাইপ' এবং 'ইন্টারফেস' উভয়ই ডেটার গঠন সংজ্ঞায়িত করতে ব্যবহত হয় ,
        *টাইপ'* : টাইপ এর ক্ষেত্রে প্রথমে টাইপ তারপর টাইপের নাম দিতে হা ,  বিভিন্ন ডাটা যেমন- ইউনিয়ন, ইন্টারসেকশন এবং সাধারণ ধরনের ডেটা (যেমনবুলিয়ান: সংখ্যা, স্ট্রিং ) সংজ্ঞায়িত করতে বেশি উপযোগী ।
        *ইন্টারফেস* :অবজেক্ট এবং ক্লাসের গঠন সংজ্ঞায়িত করা, পাবলিক API তৈরি করা। মূলত অবজেক্ট বা ক্লাসের গঠন এবং পাবলিক API সংজ্ঞায়িত করতে ব্যবহৃত হয় এবং এটি অতিরিক্ত বৈশিষ্ট্য (extended) করা যায়।
